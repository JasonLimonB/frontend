Title level 1: h1

Title level 2: h2

Title level 3: h3

Title level 4: h4

Title level 5: h5
Title level 6: h6

Other tag if you need put text is p, in this tag you can put text for paragraphs

  • Defines emphasized text, tag "em"
  • Marked text, tag "mark"
  • Smaller text, tag "small"
  • Deleted text, tag "del"
  • Inserted text, tag "ins"
  • Example Subscript text, tag "sub"
  • Example Superscript text, tag "sup"
